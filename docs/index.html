🔧 trt.Ino2Block

Arduino C++ Code → Tinkercad / Blockly XML Konverter

✨ Was macht trt.Ino2Block?

Diese WebApp konvertiert beliebigen Arduino-C++-Code mithilfe von Google Gemini KI in valides Blockly XML – das Blockprogrammierformat von Tinkercad. Keine Installation, kein Backend, läuft komplett im Browser.

⚡ Features

🤖KI-KonvertierungGoogle Gemini 1.5 Flash
🧱Blockly XMLTinkercad-kompatibel
🔍Live-VorschauBlockly im Browser
💾XML-ExportDownload als .xml
📚Beispiel-SketchesBlink, Button, Servo, Analog
🔒Key speichernLocalStorage optional
🐳Docker-readyPort 3210, nginx:alpine
🌑Dark UISchulgeeignetes Design

🚀 So funktioniert's

1
Arduino-Code einfügen

Beliebigen Sketch eingeben oder Beispiel wählen

2
Gemini API-Key eingeben

Kostenlos unter aistudio.google.com – optional speichern

3
Konvertieren klicken

KI analysiert den Code und generiert Blockly XML

4
XML exportieren oder Vorschau laden

Download als .xml oder direkte Blockly-Ansicht

🐳 Docker Schnellstart

Für Schulserver, Heimnetz oder LAN – 3 Befehle genügen:

git clone https://github.com/jbkunama1/trt.Ino2Block
cd trt.Ino2Block
docker compose up -d
# → http://<server-ip>:3210

Oder als Portainer Stack: Inhalt von docker-compose.yml einfügen – fertig.

⚡ App starten 📂 GitHub Repo 🔑 API-Key holen
