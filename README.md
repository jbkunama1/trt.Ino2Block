# 🔧 trt.Ino2Block

> Arduino C++ Code → Tinkercad/Blockly XML Konverter

## Features
- 📥 Arduino-Code eingeben oder einfügen
- ⚡ KI-gestützte Konvertierung via Google Gemini API
- 📤 Blockly XML Export als `.xml` Datei
- 🔍 Live-Vorschau der generierten Blöcke
- 📋 Kopieren in die Zwischenablage

## Voraussetzungen
- Moderner Browser (Chrome/Firefox)
- [Google Gemini API-Key](https://aistudio.google.com/app/apikey)

## Nutzung
1. Repo klonen: `git clone https://github.com/jbkunama1/trt.Ino2Block`
2. `index.html` im Browser öffnen (kein Server nötig)
3. Arduino-Code einfügen
4. Gemini API-Key eingeben
5. **Konvertieren** klicken → XML exportieren

## Tinkercad Import
1. Tinkercad → Code-Bereich öffnen
2. Blocks-Modus wählen
3. XML direkt in den Workspace einfügen

## GitHub Pages
Settings → Pages → Branch: main → App sofort online verfügbar

## Entwicklung
Reine Client-Side-App – kein Backend, kein Build-System.
